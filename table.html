DDB Dashboard

Das DDB Dashboard visualisiert einige Parameter von musealen Metadaten in der DDB. Es gibt eine Übersicht über die datengebenden Institutionen und gibt über ein Ampelsystem Aufschluss über

  • die Accessibility der Daten (welcher Anteil ist programmatisch per API abrufbar?)
  • die Normdatenabdeckung (welcher Anteil der Personen, Orte, Schlagworte ist mit kontrolliertem Vokabular versehen?)
  • die Offenheit/Lizenzierung.
Bei größeren Datengeber:innen basiert die Analyse derzeit noch auf einer Stichprobe der Datensätze.

Der Farbcode ist folgendermaßen zu interpretieren:

Kategorie grün gelb rot
Access über API abrufbar nur über Web-Interface, nicht über API abrufbar
actor, place, subject Mehr als 2/3 der Einträge normiert 1/3-2/3 der Einträge normiert weniger als 1/3 der Einträge normiert
license offen lizenziert, ohne Einschränkungen nachnutzbar eine CC-Lizenz nicht ohne Weiteres nachnutzbar, unklare Lizenz
